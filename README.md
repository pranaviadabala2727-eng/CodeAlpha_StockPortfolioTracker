# CodeAlpha Stock Portfolio Tracker

## Description

A simple Python program that calculates the total investment value of a stock portfolio based on user-entered stock names and quantities.

## Features

* Uses a dictionary to store stock prices
* Accepts stock name and quantity from the user
* Calculates total investment value
* Saves the result to a text file

## Technologies Used

* Python
* Dictionary
* Input/Output
* Arithmetic Operations
* File Handling

## How to Run

1. Open terminal or command prompt.
2. Run:
   python stock_portfolio.py
